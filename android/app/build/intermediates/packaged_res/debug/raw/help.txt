诗风 使用说明

欢迎使用 诗风！这是一款为您创作诗歌提供便捷小应用。

主要功能：
1. 点击主界面下方会打开卡片创作页面，您可以在卡片中记录您的灵感句子，并点击保存到主界面
2. 点击"词牌推荐"按钮可获取与您卡片上句子格式相似的词牌，选择该词牌为模板（也包括律诗、绝句）后可以进入诗词创作页面。
3. 点击"韵脚推荐"可输入一个字，查询与其押韵的韵脚推荐
4. 点击灵感按钮可以进入AI补全界面，按照API key申请指南填入您的deepseek API key后可以根据您选的词牌，为您卡片上的内容一键补全成完整的诗词
5. 设置中可以调节字体大小，选择字体和颜色，调节排列方向；分享中可以选择卡片背景
使用技巧：
- 您可以将您的日常灵感用卡片功能记录下来，也可以使用AI补全功能一键成诗
- 使用词牌推荐和韵脚推荐将为您创作诗词提供便捷


注意事项：
- 本应用完全保存在本地，无网络需求（AI补全功能除外）
- 当使用AI补全功能时，请确保网络连接正常以获得最佳体验
- 诗句生成需要一定时间，请耐心等待
- 每次生成的诗句都是独一无二的
- 使用AI补全功能会消耗微量费用，您可以在deepseek platform 中查看您的账单
- 按照每轮输入消耗200Tokens，输出消耗200Tokens 来看，12元可以调用deepseek-Reasoner 3,529 次,调用100次需0.34元。
